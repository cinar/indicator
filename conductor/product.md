# Project Context: Indicator\n\n## Initial Concept\nIndicator is a comprehensive Golang module designed for financial technical analysis. It provides a rich library of indicators, trading strategies, and a robust backtesting framework. The module is built for high performance, high test coverage, and flexibility through the use of Go generics and channel-based data streaming.\n\n## Product Vision\nTo be the go-to open-source Go library for financial technical analysis, offering reliable, well-tested, and performant tools for traders and developers.\n\n## Key Features\n- **Comprehensive Indicator Suite:** Support for trend, momentum, volatility, volume, and valuation indicators.\n- **Strategy Framework:** A set of base, compound, and decorator strategies for building complex trading logic.\n- **Backtesting Engine:** Evaluate strategy performance against historical asset data with visual reporting.\n- **Data Repositories:** Pluggable repository system supporting file systems, in-memory storage, and external APIs like Tiingo.\n- **High Performance:** Leverages Go channels for streaming data processing and multi-worker concurrency for synchronization and backtesting.\n- **AI-Ready:** Built-in MCP (Multi-Client Protocol Server) support for seamless integration with AI agents.\n- **Type-Safe:** Utilizes Go generics to support various numeric types while maintaining type safety.\n\n## Target Audience\n- **Traders:** Looking for reliable tools to automate their technical analysis and backtesting.\n- **Developers:** Building financial applications, trading bots, or analytical platforms in Go.\n- **Data Scientists:** Conducting research on historical market data using a robust, tested library.
